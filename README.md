# TEL330_FinalExam
NMBU, TEL330, Mechatronics final exam  
Project Overview:  
Your objective is to design, implement, and test an intelligent mechatronic system capable of tracking a salmon fillet on a moving conveyor belt. To achieve this, you will utilize a Universal Robots UR10 manipulator and an Intel RealSense camera.

As part of the hardware integration, you must design and fabricate a custom laser pointer holder. This holder must be connected to the UR10 manipulator's end-effector and continuously point a laser at the fillet as it moves, with an offset so that you can track the central feature of the provided fish fillet image (fish_fillet_black.png) provided at the end of this announcement.
